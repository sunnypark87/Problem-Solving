# Algorithm (알고리즘)

[⬅️ 메인으로 돌아가기](../README.md)

문제 해결을 위한 핵심 알고리즘 기법과 관련된 문제들을 모아둔 폴더입니다.

## 📂 하위 분류
- **BruteForce_Backtracking**: 모든 경우의 수 탐색 & 가지치기
- **Sort**: 정렬 (Quick, Merge, Topological Sort 등)
- **BinarySearch**: 이분 탐색
- **Divide_Conquer**: 분할 정복 
- **TwoPointer_SlidingWindow**: 투 포인터 & 슬라이딩 윈도우 ($O(N)$ 탐색)
- **Greedy**: 그리디 (현재 상황에서의 최적해)
- **DP**: 다이나믹 프로그래밍 (작은 문제의 해를 저장하여 재사용)
