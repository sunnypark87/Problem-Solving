# Graph Theory (그래프 이론)

[⬅️ 메인으로 돌아가기](../README.md)

노드(Node)와 간선(Edge)으로 이루어진 그래프 탐색 및 최단 경로 알고리즘과 관련된 문제들을 모아둔 폴더입니다.

## 📂 하위 분류
- **BFS_DFS**: 그래프 탐색의 기본 (너비 우선 / 깊이 우선)
- **ShortestPath**: 다익스트라(Dijkstra), 플로이드 워셜(Floyd-Warshall), 벨만 포드
- **Tree**: 트리 그래프
- **MST**: 최소 스패닝 트리 (Kruskal, Prim)
- **TopologicalSort**: 위상 정렬 (순서가 있는 작업 처리)
