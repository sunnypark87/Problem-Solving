# Data Structure (자료구조)

[⬅️ 메인으로 돌아가기](../README.md)

문제 해결을 위해 데이터를 효율적으로 저장하고 관리하는 자료구조와 관련된 문제들을 모아둔 폴더입니다.

## 📂 하위 분류
- **Linear**: 선형 자료구조 (Stack, Queue, Deque, Linked List)
- **Tree_Heap**: 트리 및 힙-우선순위 큐 (Priority Queue)
- **Map_Set**: 키-값(Key-Value) 매핑 및 집합 관리 (Hash/Tree based)
- **DisjointSet**: 유니온 파인드 (Union-Find)
